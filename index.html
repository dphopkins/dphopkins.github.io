Organic Spectra

Website (beta)

The Organic Spectra project is a web application created to help students at Haverford College learn organic spectroscopy more effectively and efficiently. Users participate in tutorials and answer timed questions about relevant material. Self-scheduled review material is available, as well as tailored material available in spaced intervals to maximize long-term memory retention. The Haverford chemistry department plans to use this in introductory chemistry classes in the near future.

You must log in to access the majority of the site. Username: Guest (case-sensitive) and Password: spectra.
